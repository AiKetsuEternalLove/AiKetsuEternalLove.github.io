.:: Frequently Asked Question ::.

  • Untuk membuat website gratis, Silahkan Bapak/Ibu buka website mysch.id lalu tekan menu BUAT SEKARANG di bagian kiri atas atau Bapak bisa langsung buka link : http://buat.mysch.id/

  • Domain resmi yang diperuntukkan untuk sekolah ber ekstensi .sch.id dan .ac.id untuk perguruan tinggi. Domain ini merupakan Top Level Domain khusus untuk sekolah, bersifat kepemilikan sementara dengan pembayaran sewa per tahun minimal 1 tahun kontrak. Persyaratan untuk membuat domain sekolah harus menyertakan surat permohonan kepala sekolah, surat kuasa kepala sekolah dan KTP kepala sekolah sehingga tidak semua orang bahkan lembaga boleh mengajukan domain ekstensi sekolah ini.

  • Jika Anda telah membuat website sendiri dengan nama domain .mysch.id dan ingin menggantinya dengan nama domain .sch.id, maka Anda hanya cukup membayara biaya upgrade sebesar Rp. 600.000,- untuk biaya sewa domain dan hosting.

  • Tidak, untuk paket website sekolah gratis dengan domain .mysch.id selamanya gratis. Sedangkan untuk website sekolah dengan domain .sch.id akan dikenai biaya perpanjangan sewa domain dan hosting sebesar Rp. 450.000/ tahun untuk Basic dan Rp. 750.000/ tahun untuk Profesional.
